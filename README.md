# 📚 Telegram Library Bot

A Telegram bot for managing a personal library using Python and SQLite.

## Features

- ➕ Add books
- 📖 View all books
- 🔍 Search books
- ✏️ Edit books
- 🗑️ Delete books
- 💾 SQLite Database
- 💬 ConversationHandler
- 🎛️ Inline Keyboard

## Technologies

- Python
- python-telegram-bot
- SQLite

## Installation

```bash
pip install -r requirements.txt
```

## Run

Replace your Telegram Bot Token inside `main.py`.

```bash
python main.py
```
